# AI Organization Guide Creator Skill

**完整工作流：** 从研究到部署的 AI 组织指南/电子书创建

---

## 📦 技能内容

| 文件 | 用途 |
|------|------|
| `SKILL.md` | 完整工作流说明 |
| `scripts/generate_chapters.py` | 章节生成脚本 |
| `references/workflow.md` | 详细流程参考 |
| `assets/cover-prompts.md` | 封面图提示词 |

---

## 🎯 功能

 encapsulates 完整的 AI 组织指南创建工作流：

```
Phase 1: 研究 (30 min)
  - 趋势分析
  - Notion 数据库
  - 产品大纲

Phase 2: 内容 (2 hours)
  - 12 章生成
  - 质量审计
  - 进度追踪

Phase 3: 设计 (30 min)
  - 销售页
  - 幻灯片
  - 封面提示词

Phase 4: 部署 (15 min)
  - GitHub Pages
  - Notion 同步
```

---

## 🔧 使用方式

### 在 OpenClaw 中调用

```
ai-organization-guide-creator: "创建 AI 变现指南电子书"
```

### 手动运行脚本

```bash
# 生成章节
python3 scripts/generate_chapters.py drafts/

# 查看工作流
cat references/workflow.md

# 获取封面提示词
cat assets/cover-prompts.md
```
