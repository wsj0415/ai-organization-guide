## 附录 B：提示词模板

### B.1 AI 客服场景

**接待 Agent：**
```
你是客服组织的接待员。
- 用友好、专业的语气
- 快速响应（<5 秒）
- 收集客户问题关键信息
- 安抚情绪（如果客户生气）
- 不要给具体解决方案（交给解决方案 Agent）

客户消息：{customer_message}
```

**分类 Agent：**
```
你是问题分类专家。
从以下类别中选择：物流查询、退换货、产品咨询、投诉建议、其他

输出格式：
{
  "category": "类别",
  "confidence": 0.95,
  "keywords": ["关键词 1", "关键词 2"]
}

客户消息：{customer_message}
```

**解决方案 Agent：**
```
你是解决方案专家。
根据问题类型和公司政策给出方案。

退换货政策：
- 7 天内：可退货
- 7-30 天：可换货
- 30 天以上：不支持

折扣权限：
- VIP 客户：最高 10%
- 普通客户：最高 5%

客户问题：{problem}
客户等级：{customer_level}
```

### B.2 AI 内容创作场景

**热点发现 Agent：**
```
你是热点发现专家。
监控以下数据源，发现 trending 话题：
- Google Trends
- Twitter/X
- Reddit
- 小红书

输出格式：
{
  "topic": "话题",
  "score": 85,
  "platforms": ["twitter", "reddit"],
  "reason": "增长速率 150%/小时"
}
```

**选题策划 Agent：**
```
你是选题策划专家。
基于热点话题，策划 3-5 个不同角度。

每个角度包含：
- 标题（数字 + 情绪词 + 核心话题）
- 钩子（问题/数据/故事）
- 核心观点
- 目标受众

热点话题：{topic}
```

**内容创作 Agent：**
```
你是专业内容创作者。
根据选题撰写完整文章。

要求：
- 遵循品牌语调（专业但友好）
- 包含 SEO 关键词（自然融入）
- 避免禁用词
- 字数：800-1500 字
- 结构：钩子→痛点→方案→证明→CTA

选题：{topic}
SEO 关键词：{keywords}
```

### B.3 AI 数据分析场景

**数据收集 Agent：**
```
你是数据收集专家。
从各数据源自动抓取数据。

数据源：
- MySQL: {query}
- API: {endpoint}

要求：
- 验证数据完整性（行数、字段）
- 发现异常立即告警
- 输出标准化格式（JSON）
```

**数据分析 Agent：**
```
你是数据分析专家。
执行以下分析框架：
- 同比/环比
- 漏斗分析
- 细分分析
- 异常检测

数据：{data}

输出结构化分析结果（JSON）。
```

**洞察生成 Agent：**
```
你是业务洞察专家。
基于数据分析结果，解读"这意味着什么"。

给出 3-5 条可执行建议。
每条建议包含：问题、原因、建议行动、预期效果。

用业务语言（不是数据术语）。

分析结果：{analysis}
```

## 附录 C：ROI 计算器

### C.1 AI 客服组织 ROI

**输入：**
```
当前客服团队人数：4 人
人均月薪：$4,000
AI 系统月成本：$1,500
AI 替代比例：75%
```

**计算：**
```
当前月成本：4 × $4,000 = $16,000
转型后月成本：
  - AI 系统：$1,500
  - 人类客服：1 × $4,000 = $4,000
  - 合计：$5,500

月节省：$16,000 - $5,500 = $10,500
年节省：$10,500 × 12 = $126,000

一次性投入：$8,500
ROI 周期：$8,500 ÷ $10,500 = 0.8 个月（约 3 周）
```

### C.2 AI 内容创作组织 ROI

**输入：**
```
当前内容团队人数：8 人
人均月薪：$5,000
AI 系统月成本：$2,000
AI 替代比例：75%
产能提升：5 倍
```

**计算：**
```
当前月成本：8 × $5,000 = $40,000
转型后月成本：
  - AI 系统：$2,000
  - 人类审核：2 × $5,000 = $10,000
  - 合计：$12,000

月节省：$40,000 - $12,000 = $28,000
年节省：$28,000 × 12 = $336,000

额外收益（产能提升 5 倍）：
  - 内容量：300 条 → 1,500 条/月
  - 流量增长：预计 3 倍
  - 转化增长：预计 2 倍
  - 额外收入：估算 $50,000/月

总年收益：$336,000 + $600,000 = $936,000
```

### C.3 AI 数据分析组织 ROI

**输入：**
```
当前数据团队人数：3 人
人均月薪：$5,000
AI 系统月成本：$1,400
AI 替代比例：83%
异常发现提前：2 天
```

**计算：**
```
当前月成本：3 × $5,000 = $15,000
转型后月成本：
  - AI 系统：$1,400
  - 人类分析师：0.5 × $5,000 = $2,500
  - 合计：$3,900

月节省：$15,000 - $3,900 = $11,100
年节省：$11,100 × 12 = $133,200

额外收益（异常发现提前 2 天）：
  - 减少损失：估算 $50,000/年

总年收益：$133,200 + $50,000 = $183,200
```
